Kinetis MKW37A MKW38A and MKW39A Wireless Dual Mode (Bluetooth LE and IEEE® 802.15.4) Microcontrollers Low-power Library Overview

Low-power library for MKW37/38/39 is the same as the one for Kinetis Wireless Dual Mode Microcontrollers in Kinetis Wireless Dual Mode Microcontrollers Low-power Library Overview. It is enhanced with application dedicated low-power modes.

  • Deep sleep mode 4
