Deep sleep modes

Six low-power modes have been developed for the connectivity core and four low power modes for the application core. To maintain compatibility to previous low power libraries and applications, the implementation contains 7 power modes for each core. The user can switch between them at run-time using PWR_ChangeDeepSleepMode function on each core but it is recommendable do this from the application core only. PWR_ChangeBlackBoxDeepSleepMode function can be called by the application code to control the low power mode on the connectivity core. The default low-power mode for each core is selected by the cPWR_DeepSleepMode define value in the core’s dedicated PWR_Configuration.h header file.
