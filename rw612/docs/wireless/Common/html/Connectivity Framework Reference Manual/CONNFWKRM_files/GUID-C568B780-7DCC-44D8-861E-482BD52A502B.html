Application dedicated low-power modes

Three low-power modes were defined to reflect application connectivity state (No activity, Advertising, Connected), then the low-power module can match to the most appropriate device low-power mode for each mode. The API PWR_SetNewAppState(...) is used to reflect Application connectivity state.
