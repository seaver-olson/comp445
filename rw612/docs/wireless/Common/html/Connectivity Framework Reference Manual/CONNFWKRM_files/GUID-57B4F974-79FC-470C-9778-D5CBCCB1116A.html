Configuration Macros

Name:

#define gDCDC_Enabled_d 0

Description:

Enables/disables DCDC-related code and variables.

Location:

DCDC.h

Name:

#define APP_DCDC_MODE gDCDC_Buck_c

Description:

Set this define to gDCDC_Bypass_c if DCDC is in bypass mode or gDCDC_Buck_c. It configures deep sleep mode 4 to put the MCU in VLLS0 or VLLS1 because VLLS0 is only supported in bypass mode.

Location:

app_preinclude.h

Name:

#define gApp_EnableDCDCShutdown_d (1)

Description:

Setting this define allows DCDC shutdown on Pswitch pin.

Location:

app_preinclude.h
