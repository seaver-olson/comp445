Deep sleep mode 4

Deep sleep mode 4 is the lowest power mode. The MCU enters mode 4 when both cores have entered mode 4.

The application core enters VLLS0/VLLS1 in this mode and no RAM is retained. No synchronization for low-power timers is made because this deep sleep mode is exited through reset sequence.

For the connectivity core, this mode is used when all stacks enabled for this platform are idle. In this mode, the cm0+ enters VLLS0/VLLS1 and all enabled link layers remain idle. No synchronization for low-power timers is made because this deep sleep mode is exited through reset sequence.

The MCU wakes up from this mode by the LLWU connected onboard switches or by reset.
