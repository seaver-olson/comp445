Deep sleep mode 8

Deep sleep mode 8 is the lowest power mode which allows RAM retention. It was designed for those applications which require RAM retention over low power. In this mode, the MCU enters VLLS2 or VLLS3 depending on the amount of RAM retained. All stacks enabled to run on this platform remain idle. In VLLS3 all the RAM is retained. In VLLS2, only 32 KB RAM is retained. It depends on Cstack address defined in linker project config : gCstackTopAddr_c. When gCstackTopAddr_c is less than 0x20008000 , VLLS2 is applied and only 32 KB RAM is retained. Otherwise, VLLS3 is applied and all 64 KB RAM is retained. In this mode, Bluetooth Low Energy Link Layer wake-up instant is 3 slots earlier compared to other low-power modes.
